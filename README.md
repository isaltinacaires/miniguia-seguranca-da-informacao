# Miniguia de Estudos - Segurança da Informação com NotebookLM

## 📌 Contexto

Este projeto foi desenvolvido como desafio prático do Bootcamp Santander na DIO.

O objetivo foi utilizar Inteligência Artificial como ferramenta de aprendizagem ativa, organizando conteúdos no NotebookLM para criar um caderno temático sobre Segurança da Informação e Criptografia.

---

## 🎯 Objetivos de Estudo

- Compreender os princípios da Segurança da Informação;
- Estudar os conceitos de confidencialidade, integridade e disponibilidade;
- Entender os principais tipos de criptografia;
- Aprender sobre ameaças cibernéticas e formas de proteção;
- Utilizar IA para resumir conteúdos e auxiliar nos estudos.
